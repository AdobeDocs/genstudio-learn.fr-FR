# Métadonnées pour utilisation interne

Le fichier metadata.md comprend des métadonnées de niveau référentiel qui transitent par les fichiers TOC.md du guide de l’utilisateur dans le référentiel. Si vous souhaitez modifier le contenu de metadata.md pour n’importe quel guide de l’utilisateur, faites-le dans n’importe quel fichier TOC.md.

Consultez les [Métadonnées](https://experienceleague.adobe.com/docs/authoring-guide-exl/using/editing/user-guide-setup/metadata.html?lang=fr) pour obtenir une liste de valeurs valides pour la **solution**, le **produit** et le **type**.
